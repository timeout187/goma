Both problem types render an illustrative surface shaped by the sliders above — they are not connected to Goma's actual Newton/FEM solver or to real Exodus output. They're a stand-in for the "see the physics move" GUI, covering two of the classic free/moving-boundary problem classes Goma solves (thin-film coating flows, and wetting/contact-angle problems), built to sit alongside the real program while a proper results viewer (reading actual Goma output files) is scoped. See BUILD.md for building the real solver.
